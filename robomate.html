Robomate

Project Specification

A prototype personal assistant robot combining affordability with efficiency. Features a speech-enabled assistant for daily tasks, autonomous and remote-controlled navigation, and colour recognition that triggers visual reactions on its display.

Role
Programmer
Team Size
2 Students
Hardware
Raspberry Pi
Language
Python
My Contributions
  • Configured Raspberry Pi by installing Alexa and Mycroft, enabling voice assistant functionality
  • Developed movement scripts for remote control via tablet or computer
  • Implemented collision avoidance algorithms for autonomous navigation
  • Assembled hardware components: main body, ultrasonic sensors, camera, speakers, motors, display, and microphone
  • Programmed colour recognition to identify three colours and display corresponding facial expressions
